# Bash –∏ —Ä–µ–≥—É–ª—è—Ä–Ω—ã–µ –≤—ã—Ä–∞–∂–µ–Ω–∏—è

–í –ø—Ä–æ—à–ª—ã–π —Ä–∞–∑ –º—ã —Ä–∞—Å—Å–º–æ—Ç—Ä–µ–ª–∏ —Å–ø–æ—Å–æ–± –∞–≤—Ç–æ–º–∞—Ç–∏–∑–∏—Ä–æ–≤–∞—Ç—å —Ä—É—Ç–∏–Ω–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å–±–æ—Ä–∫–∏: Make. –ù–æ –ø–æ–Ω—è—Ç–Ω–æ, —á—Ç–æ —ç—Ç–æ –Ω–µ –µ–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω–∞—è –ø—Ä–µ–¥–º–µ—Ç–Ω–∞—è –æ–±–ª–∞—Å—Ç—å, –≤ –∫–æ—Ç–æ—Ä–æ–π –º–æ–∂–Ω–æ —á—Ç–æ-—Ç–æ –∞–≤—Ç–æ–º–∞—Ç–∏–∑–∏—Ä–æ–≤–∞—Ç—å. –° –ø–æ–º–æ—â—å—é —Å–∫—Ä–∏–ø—Ç–æ–≤ –º–æ–∂–Ω–æ –Ω–∞—Å—Ç—Ä–∞–∏–≤–∞—Ç—å –æ–∫—Ä—É–∂–µ–Ω–∏–µ –¥–ª—è —Ä–∞–∑—Ä–∞–±–æ—Ç–∫–∏, http –∑–∞–ø—Ä–æ—Å—ã –¥–ª—è –Ω–∞–≥—Ä—É–∑–æ—á–Ω–æ–≥–æ —Ç–µ—Å—Ç–∏—Ä–æ–≤–∞–Ω–∏—è, —á–∏—Å—Ç–∏—Ç—å —Å—Ç–∞—Ä—ã–µ –ª–æ–≥–∏ –∏ –∞–≤—Ç–æ–º–∞—Ç–∏–∑–∏—Ä–æ–≤–∞—Ç—å –º–Ω–æ–∂–µ—Å—Ç–≤–æ –¥—Ä—É–≥–∏—Ö –≤–µ—â–µ–π.

–°–µ–≥–æ–¥–Ω—è –º—ã –ø–æ–≥–æ–≤–æ—Ä–∏–º –æ bash. –≠—Ç–æ —Å–∫—Ä–∏–ø—Ç–æ–≤—ã–π —è–∑—ã–∫, –∫–æ—Ç–æ—Ä—ã–π –¥–æ—Å—Ç—É–ø–µ–Ω –∏–∑ –∫–æ—Ä–æ–±–∫–∏ –≤ –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–µ UNIX (–∏ –Ω–µ —Ç–æ–ª—å–∫–æ) —Å–∏—Å—Ç–µ–º. –¢–µ—Ä–º–∏–Ω–∞–ª —ç—Ç–∏—Ö —Å–∏—Å—Ç–µ–º —è–≤–ª—è–µ—Ç—Å—è –∏–Ω—Ç–µ—Ä–∞–∫—Ç–∏–≤–Ω–æ–π –æ–±–æ–ª–æ—á–∫–æ–π –Ω–∞–¥ –Ω–∏–º. –§–∞–π–ª—ã —Å–æ —Å–∫—Ä–∏–ø—Ç–∞–º–∏ –æ–±—ã—á–Ω–æ –∏–º–µ—é—Ç —Ä–∞—Å—à–∏—Ä–µ–Ω–∏–µ ``sh``.

–°–∫–æ—Ä–µ–µ –≤—Å–µ–≥–æ –≤—ã —É–∂–µ –∑–Ω–∞–∫–æ–º—ã —Å –¥—Ä—É–≥–∏–º —Å–∫—Ä–∏–ø—Ç–æ–≤—ã–º —è–∑—ã–∫–æ–º -- Python. –ú–æ–∂–µ—Ç —Å—Ä–∞–∑—É –≤–æ–∑–Ω–∏–∫–Ω—É—Ç—å –∑–∞–∫–æ–Ω–æ–º–µ—Ä–Ω—ã–π –≤–æ–ø—Ä–æ—Å, –∫–æ–≥–¥–∞ –ª—É—á—à–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å bash. –ü–æ –º–æ–µ–º—É –æ–ø—ã—Ç—É –æ–Ω —É–¥–æ–±–µ–Ω, –∫–æ–≥–¥–∞ –Ω—É–∂–Ω–æ –Ω–∞–ø–∏—Å–∞—Ç—å –Ω–µ–∫–æ—Ç–æ—Ä—É—é –æ–±–≤—è–∑–∫—É –Ω–∞–¥ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–º–∏ —É—Ç–∏–ª–∏—Ç–∞–º–∏ –ø–æ —Ç–∏–ø—É ``head``, ``ls``, ``grep``. –í –æ—Å—Ç–∞–ª—å–Ω—ã—Ö —Å–ª—É—á–∞—è—Ö —è –ø—Ä–µ–¥–ø–æ—á–∏—Ç–∞—é —Å—Ä–∞–∑—É –ø–∏—Å–∞—Ç—å –Ω–∞ Python.

## –ö–æ–º–∞–Ω–¥—ã

–í—ã —É–∂–µ –Ω–∞–≤–µ—Ä–Ω—è–∫–∞ —Å—Ç–∞–ª–∫–∏–≤–∞–ª–∏—Å—å —Å —Ç–µ—Ä–º–∏–Ω–∞–ª–æ–º –≤ –ø—Ä–æ—à–ª—ã—Ö –∫—É—Ä—Å–∞—Ö –∏ –ø–æ–ª—å–∑–æ–≤–∞–ª–∏—Å—å, –Ω–∞–ø—Ä–∏–º–µ—Ä, –∫–æ–º–∞–Ω–¥–æ–π ``ls``. –ù–∞ —Å–∞–º–æ–º –¥–µ–ª–µ, –µ—Å—Ç—å –¥–≤–∞ —Ç–∏–ø–∞ –∫–æ–º–∞–Ω–¥: –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã–µ –≤ –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ç–æ—Ä –∏ –≤–Ω–µ—à–Ω–∏–µ. –ö –ø–µ—Ä–≤—ã–º –æ—Ç–Ω–æ—Å–∏—Ç—Å—è, –Ω–∞–ø—Ä–∏–º–µ—Ä, ``cd`` (—Å–ø–∏—Å–æ–∫ –≤—Å–µ—Ö —Ç–∞–∫–∏—Ö –∫–æ–º–∞–Ω–¥ –º–æ–∂–Ω–æ –ø–æ—Å–º–æ—Ç—Ä–µ—Ç—å –≤ ``man builtins``), –∫–æ –≤—Ç–æ—Ä—ã–º ``ls``. –í–Ω–µ—à–Ω–∏–µ –ø—Ä–æ–≥—Ä–∞–º–º—ã –∏—â—É—Ç—Å—è –ø–æ –ø–æ—Ä—è–¥–∫—É –ø–æ –ø—É—Ç—è–º, –ø–µ—Ä–µ—á–∏—Å–ª–µ–Ω–Ω—ã–º –≤ –≥–ª–æ–±–∞–ª—å–Ω–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π ``$PATH``.

> –ï—Å–ª–∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ –ø–æ–Ω—è—Ç—å, –≥–¥–µ –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ã–π —Ñ–∞–π–ª, –º–æ–∂–Ω–æ –≤–æ—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –∫–æ–º–∞–Ω–¥–æ–π ``which``.
> –ù–∞–ø—Ä–∏–º–µ—Ä: ``which whoami``.
> –ë–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ –∏–∑–≤–µ—Å—Ç–Ω—ã—Ö –≤–∞–º —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã—Ö —É—Ç–∏–ª–∏—Ç –Ω–∞—Ö–æ–¥—è—Ç—Å—è –≤ ``/usr/bin``.

–° —Ç–æ—á–∫–∏ –∑—Ä–µ–Ω–∏—è —Ä–∞—Å—à–∏—Ä—è–µ–º–æ—Å—Ç–∏ –±—ã–ª–æ –±—ã –ª–æ–≥–∏—á–Ω–æ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å –≤—Å–µ –∫–æ–º–∞–Ω–¥, –æ–¥–Ω–∞–∫–æ —É —ç—Ç–æ–≥–æ –ø–æ–¥—Ö–æ–¥–∞ –µ—Å—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –ø—Ä–æ–±–ª–µ–º:

* –ó–∞–ø—É—Å–∫ –≤–Ω–µ—à–Ω–∏—Ö –ø—Ä–æ–≥—Ä–∞–º–º –º–µ–¥–ª–µ–Ω–Ω–µ–µ
* –í–Ω–µ—à–Ω–∏–µ –∫–æ–º–∞–Ω–¥—ã –Ω–µ –º–æ–≥—É—Ç –ø–æ–≤–ª–∏—è—Ç—å –Ω–∞ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ –∫–æ–º–∞–Ω–¥–Ω–æ–π –æ–±–æ–ª–æ—á–∫–∏

## –ë–∞–∑–æ–≤—ã–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å

Bash —Å–∫—Ä–∏–ø—Ç—ã –æ–±—ã—á–Ω–æ –ø—Ä–∏–Ω—è—Ç–æ –ø–∏—Å–∞—Ç—å –≤ —Ñ–∞–π–ª–∞—Ö —Å —Ä–∞—Å—à–∏—Ä–µ–Ω–∏–µ–º ``.sh``. –ö–∞–∫ –∏ –≤ —Å–ª—É—á–∞–µ –¥—Ä—É–≥–∏—Ö —Å–∫—Ä–∏–ø—Ç–æ–≤—ã—Ö —è–∑—ã–∫–æ–≤, —á—Ç–æ–±—ã –∑–∞–ø—É—Å–∫–∞—Ç—å —Å–∫—Ä–∏–ø—Ç —á–µ—Ä–µ–∑ ``./script_name``, –Ω—É–∂–Ω–æ –Ω–µ –∑–∞–±—ã—Ç—å —Å–¥–µ–ª–∞—Ç—å –¥–≤–µ –≤–µ—â–∏:

* –£–∫–∞–∑–∞—Ç—å –≤ –Ω–∞—á–∞–ª–µ —Ñ–∞–π–ª–∞ –ø—É—Ç—å –∫ –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ç–æ—Ä—É —è–∑—ã–∫–∞. –í –Ω–∞—à–µ–º —Å–ª—É—á–∞–µ —ç—Ç–æ ``#!/usr/bin/bash``
* –°–¥–µ–ª–∞—Ç—å —Ñ–∞–π–ª –∏—Å–ø–æ–ª–Ω—è–µ–º—ã–º. –ù–∞–ø—Ä–∏–º–µ—Ä —Å –ø–æ–º–æ—â—å—é –∫–æ–º–∞–Ω–¥—ã ``chmod +x script_name``.

## –ü–µ—Ä–µ–º–µ–Ω–Ω—ã–µ

* –û–±—ä—è–≤–∏—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é: ``foo=bar``. –ü–æ —É–º–æ–ª—á–∞–Ω–∏—é –≤—Å–µ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –≥–ª–æ–±–∞–ª—å–Ω—ã–µ –∏ –¥–æ—Å—Ç—É–ø–Ω—ã —Å –º–æ–º–µ–Ω—Ç–∞ –æ–±—ä—è–≤–ª–µ–Ω–∏—è. ``local foo=bar`` –ø–æ–∑–≤–æ–ª—è–µ—Ç –æ–±—ä—è–≤–∏—Ç—å –ª–æ–∫–∞–ª—å–Ω—É—é –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é (–ø–æ–ª–µ–∑–Ω–æ, –Ω–∞–ø—Ä–∏–º–µ—Ä, –≤–Ω—É—Ç—Ä–∏ —Ñ—É–Ω–∫—Ü–∏–∏)
* –ü–æ–ª—É—á–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ: ``$foo``. –ú–æ–∂–Ω–æ –ø–æ–¥—Å—Ç–∞–≤–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –≤ —Å—Ç—Ä–æ–∫—É, –Ω–æ —Ç–æ–≥–¥–∞ –æ–Ω–∞ –¥–æ–ª–∂–Ω–∞ –±—ã—Ç—å –∑–∞–∫–ª—é—á–µ–Ω–∞ –≤ –¥–≤–æ–π–Ω—ã–µ –∫–∞–≤—ã—á–∫–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, ``echo "Score: $score``).
* –ó–∞–ø–∏—Å–∞—Ç—å —Ä–µ–∑—É–ª—å—Ç–∞—Ç –∫–æ–º–∞–Ω–¥—ã –≤ –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é: ``foo=$(command)``. 

> –ù–µ–ª—å–∑—è —Å—Ç–∞–≤–∏—Ç—å –ø—Ä–æ–±–µ–ª—ã –≤–æ–∫—Ä—É–≥ –∑–Ω–∞–∫–æ –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏–π, –ø–æ—Ç–æ–º—É —á—Ç–æ —Å —Ç–æ—á–∫–∏ –∑—Ä–µ–Ω–∏—è –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ç–æ—Ä–∞ —ç—Ç–æ –±—É–¥–µ—Ç –≤—ã–∑–æ–≤ –∫–æ–º–∞–Ω–¥—ã `foo` —Å –∞—Ä–≥—É–º–µ–Ω—Ç–∞–º–∏ `=` –∏ `bar`.

–í –∫–∞—á–µ—Å—Ç–≤–µ –ø—Ä–∏–º–µ—Ä–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö –º–æ–∂–Ω–æ –ø–æ—Å–º–æ—Ç—Ä–µ—Ç—å –Ω–∞ —Å–∫—Ä–∏–ø—Ç ``greet_me.sh``.

–í —Å–ª—É—á–∞–µ –º–∞—Å—Å–∏–≤–æ–≤:

* –û–±—ä—è–≤–ª–µ–Ω–∏–µ: ``$foo=(alex ivan vasya)``
* –ö–æ–ª–∏—á–µ—Å—Ç–≤–æ —ç–ª–µ–º–µ–Ω—Ç–æ–≤: ``${#foo[@]}``
* –î–æ—Å—Ç—É–ø –ø–æ –∏–Ω–¥–µ–∫—Å—É: ``${foo[$i]}``

## –ê—Ä–≥—É–º–µ–Ω—Ç—ã

–ê—Ä–≥—É–º–µ–Ω—Ç—ã —Å–∫—Ä–∏–ø—Ç–∞ –ø—Ä–∏–Ω—è—Ç–æ —Ä–∞–∑–¥–µ–ª—è—Ç—å –ø—Ä–æ–±–µ–ª–∞–º–∏. –ï—Å–ª–∏ –∞—Ä–≥—É–º–µ–Ω—Ç —Å–æ–¥–µ—Ä–∂–∏—Ç –ø—Ä–æ–±–µ–ª, –º–æ–∂–Ω–æ –∑–∞–∫–ª—é—á–∏—Ç—å –µ–≥–æ –≤ –∫–∞–≤—ã—á–∫–∏ —Ü–µ–ª–∏–∫–æ–º, –∏–ª–∏ –ø–æ—Å—Ç–∞–≤–∏—Ç—å `\` –Ω–µ–ø–æ—Å—Ä–µ–¥—Å—Ç–≤–µ–Ω–Ω–æ –ø–µ—Ä–µ–¥ –ø—Ä–æ–±–µ–ª–æ–º, —á—Ç–æ–±—ã –µ–≥–æ —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞—Ç—å. –û–±—ã—á–Ω—ã–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã –Ω–∞–∑—ã–≤–∞—é—Ç—Å—è –ø–æ–∑–∏—Ü–∏–æ–Ω–Ω—ã–º–∏, –±—ã–≤–∞—é—Ç –µ—â—ë –æ–ø—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–µ, –∏—Ö –∏–º–µ–Ω–∞ –Ω–∞—á–∏–Ω–∞—é—Ç—Å—è —Å `-` –∏–ª–∏ ``--``.

–í–æ–∑—å–º—ë–º –¥–ª—è –ø—Ä–∏–º–µ—Ä–∞ –∫–æ–º–∞–Ω–¥—É ``ls -l dir_name``. –ó–¥–µ—Å—å ``dir_name`` –ø–æ–∑–∏—Ü–∏–æ–Ω–Ω—ã–π –∞—Ä–≥—É–º–µ–Ω—Ç, –∞ ``-l`` –æ–ø—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–π.

–° –∞—Ä–≥—É–º–µ–Ω—Ç–∞–º–∏ –º–æ–∂–Ω–æ —Ä–∞–±–æ—Ç–∞—Ç—å —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:

* ``$0..$9`` –ø–æ–ª—É—á–∏—Ç—å –∞—Ä–≥—É–º–µ–Ω—Ç –ø–æ –Ω–æ–º–µ—Ä—É, –≤ ``$0`` –±—É–¥–µ—Ç –ª–µ–∂–∞—Ç—å –∏–º—è —Å–∫—Ä–∏–ø—Ç–∞.
* ``$#`` –ø–æ–ª—É—á–∏—Ç—å –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤
* ``$@`` –ø–æ–ª—É—á–∏—Ç—å —Å–ø–∏—Å–æ–∫ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤.

–¢–µ–ø–µ—Ä—å –º—ã –º–æ–∂–µ–º –ø–æ–ø—Ä–∏–≤–µ—Ç—Å–æ–≤–∞—Ç—å –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è, –∫–æ—Ç–æ—Ä—ã–π –ø–µ—Ä–µ–¥–∞—Å—Ç –Ω–∞–º —Å–≤–æ—ë –∏–º—è (—Å–º. ``greet_one.sh``).

## –¶–∏–∫–ª—ã

–û—Å–Ω–æ–≤–Ω–æ–π —Å–ø–æ—Å–æ–± –æ–±—ä—è–≤–ª–µ–Ω–∏—è —Ü–∏–∫–ª–∞ –≤—ã–≥–ª—è–¥–∏—Ç —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:

```
for item in list
do
    commands
done
```

–í–æ–æ—Ä—É–∂–∏–≤—à–∏—Å—å —ç—Ç–∏–º –∑–Ω–∞–Ω–∏–µ–º, –º–æ–∂–µ–º –Ω–∞—É—á–∏—Ç—å—Å—è –ø—Ä–∏–≤–µ—Ç—Å—Ç–≤–æ–≤–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö –ª—é–¥–µ–π (—Å–º. ``greet_many.c``).

–ö–ª–∞—Å—Å–∏—á–µ—Å–∫–∏–π –≤–∞—Ä–∏–∞–Ω—Ç —Å —á–∏—Å–ª–∞–º–∏, –º–æ–∂–Ω–æ —Å–¥–µ–ª–∞—Ç—å —Å –ø–æ–º–æ—â—å—é –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä–∞ —Å–ø–∏—Å–∫–∞ ``{from..to..step}``, –Ω—É –∏–ª–∏ –ø—Ä–æ—Å—Ç–æ –≤–æ—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–Ω—ã–º —Å–∏–Ω—Ç–∞–∫—Å–∏—Å–æ–º (–Ω–æ —ç—Ç–æ –±—É–¥–µ—Ç —Ä–∞–±–æ—Ç–∞—Ç—å —Ç–æ–ª—å–∫–æ –≤ ``bash`` –∏ ``zsh``).

```
for (( i=0; i<10; i++ ))
do
   echo "$i"
done
```

–¶–∏–∫–ª ``while`` —Ç–∞–∫–∂–µ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç:

```
while condition
do
    commands
done
```

## –£—Å–ª–æ–≤–∏—è

–£—Å–ª–æ–≤–∏—è –æ–±—ä—è–≤–ª—è—é—Ç—Å—è —Å –ø–æ–º–æ—â—å—é ``[[ condition ]]`` (–ø—Ä–æ–±–µ–ª—ã –≤–∞–∂–Ω—ã, –ø–æ—Ç–æ–º—É —á—Ç–æ ``[[`` —ç—Ç–æ –Ω–∞ —Å–∞–º–æ–º –¥–µ–ª–µ –∏–º—è –∫–æ–º–∞–Ω–¥—ã).

–ú–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å:
* –°—Ä–∞–≤–Ω–µ–Ω–∏—è —á–∏—Å–µ–ª: ``[[ $x -eq $y ]]``. (—Ç–∞–∫–∂–µ –¥–æ—Å—Ç—É–ø–Ω—ã ``-lt``, ``-ne`` –∏ —Ç. –¥.).
* –°—Ä–∞–≤–Ω–µ–Ω–∏—è —Å—Ç—Ä–æ–∫: ``[[ $str1=$str2 ]]``
* –ü—Ä–æ–≤–µ—Ä–∫—É –Ω–∞ –ø—É—Å—Ç–æ—Ç—É —Å—Ç—Ä–æ–∫ ``[[ -z $str ]]`` (``-n`` –Ω–µ–ø—É—Å—Ç–æ—Ç—É)
* –õ–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã: ``[[ $cond1&&$cond2 ]]``
* –°—É—â–µ—Å—Ç–≤–æ–≤–∞–Ω–∏–µ –ø—É—Ç–∏: ``[[ -e filename ]]`` (``-f`` –æ–±—ã—á–Ω—ã–π —Ñ–∞–π–ª, ``-d`` –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏—è, ``-e`` –∏—Å–ø–æ–ª–Ω—è–µ–º—ã–π)

## sed
